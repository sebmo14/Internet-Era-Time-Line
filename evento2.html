Adopción del protocolo TCP/IP como estándar de comunicación.

1983

En 1983, el protocolo TCP/IP fue adoptado como el estándar para todas las redes conectadas a ARPANET. Esto permitió que diferentes tipos de redes se comunicaran entre sí, sentando las bases para el Internet global tal como lo conocemos hoy.
